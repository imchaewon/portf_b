About Me

안녕하세요. 백엔드 웹개발자를 희망하는 임채원입니다.
저는 웹에이전시 회사에서 웹퍼블리싱 업무를 맡다가, 뒤에서 고군분투하며 사이트제작에 큰 기여를 하는 백엔드개발에 대해 호기심을 느끼고 보다 더 깊은 바다로 나아가고자 전역후에 개발자 양성 과정을 이수하게 되었습니다.

웹을 제작하기 위해서는 코딩만 하는것이 아닌 협업을 하기 위해 많은 회의와 소통이 필요합니다.
항상 열린 생각으로 타인의 의견을 수용하며 함께 지식을 공유하고, 나만 알아볼 수 있게 만들기 보다 어떤 사람이 봐도 한눈에 알아볼 수 있는 서비스를 개발하겠습니다. 계속해서 미래를 지향하는 정신으로 어떻게 해야 보다 효율적인 프로그램을 구현할 수 있을지 늘 궁리하고 발전하는 개발자로 거듭나고자 합니다.

< 이용한 기술 >


spring framework를 이용한 의존성 주입과 3 Tier 구조를 기반으로 프로젝트를 진행하였으며,
MyBatis를 이용해 DB를 서버와 교류할 수 있는 환경을 세팅했습니다.

PL/SQL 프로시저 - 조건에 의한 UPDATE와 INSERT 처리를 구현했습니다(회원추가정보 추가 / 회원 정보 수정 / 회원탈퇴).
PL/SQL 트리거 - Delete가 발생할시 백업 테이블 생성되도록 하였습니다(회원테이블 / 달력테이블).
Ajax - 비동기처리와 JSON포맷을 이용한 정보 송신/수신을 구현했습니다(아이디 중복체크, 커뮤니티 등).
정규표현식 - 정규식을 이용해 유효성검사를 처리했습니다(회원가입, 내 체형 알아보기, 정보 수정 등)
@Transactional – Mybatis의 delete와 insert를 트랜잭션처리해 삽입에 실패했다면 데이터를 이전상태로 원복되도록 처리했습니다(운동루틴).
Request/Resonse/Model를 이용해 클라이언트/서버간 데이터 통신을 구현했습니다.
페이지 객체와 힌트를 이용한 페이징 처리를 구현했습니다(게시판).
Session/Cookie - Session과 Cookie를 이용한 로그인과 자동로그인 기능을 구현했습니다.
JSTL(조건/반복문/데이터포맷), EL – 서버로부터 받은 정보를 클라이언트에서 받아올 때 이용했습니다.
카카오맵 api - api를 이용해 지도에 위치 표시하기를 구현했습니다.

문의게시판 조회/작성/수정/삭제 회원은 문의게시판의 본인 글과 공개글을 조회할 수 있으며 글 작성, 수정, 삭제할 수 있다.
